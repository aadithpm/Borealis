About

Borealis is a free, open-source CSS framework. With only ~6 kb minified, you have space for all your other files. This framework makes it so that you don't have to type any css yourself! All you have to do is make sure that you use the correct classes.

Open Source

Borealis is 100% open source. If you find anything that you would like to change, visit the repository and open a pull request. If you would like to use this simple CSS framework, the directions are in the repository.
